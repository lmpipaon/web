Privacy Policy

Last updated: January 16, 2026

This Privacy Policy describes how ThingSpeak Visualizer, developed by Luis Pipaon, handles information on the user's device.

1. Information Collection and Storage

Our application is designed with a "Privacy by Design" approach. We do not collect, sell, or share any personally identifiable information (PII) on external servers.

  • Configuration Data: Technical identifiers such as Channel IDs and API Keys required for the app to function are stored strictly locally on the user's device using secure storage methods.
  • User Preferences: Language selection (supporting up to 9 languages) and interface customization settings are saved locally to enhance the user experience.

2. Internet Usage and Third-Party Services

The application requires Internet access exclusively for the following purposes:

  • ThingSpeak Integration: The app connects directly to ThingSpeak servers to fetch the data required to generate visual charts.
  • No Tracking: We do not use third-party analytics, tracking tools, or advertising networks that collect data from your device.

3. Data Security

Since the application manages data locally, the security of the stored information depends on the security of the user's device. All external data transmissions to ThingSpeak are performed over encrypted channels (HTTPS).

4. Multi-language Support

The application offers support for multiple languages. This configuration is handled internally and does not involve sending location data or cultural preferences to any external entity.
